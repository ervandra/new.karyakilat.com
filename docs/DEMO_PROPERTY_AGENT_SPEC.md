## Design System

### Colors
- Background: `#0a0a0a` (near-black)
- Card bg: `#111111`
- Border: `rgba(255,255,255,0.08)`
- Primary/Gold: `#d4af37` or `#c9a84c`
- Text: `#f5f5f5` (headings), `#a0a0a0` (body/muted)

### Typography
- Headings: Serif font (Playfair Display or similar â€” use next/font/google)
- Body: Sans-serif (Inter)
- Mix: Serif for names/headlines, sans for body text

### Feel
- "Silent Luxury" â€” minimal, dark, gold accents
- No gradients that feel cheap (subtle dark-to-darker gradients only)
- Large whitespace, breathable layout
- Gold line separators instead of hard borders where possible
